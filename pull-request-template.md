## 📌 PR 제목 예시
[WeekN][이름] 작업 요약

---

## 1. 작업 내용
- 어떤 기능/수정/리팩터링을 했는지 간단히 설명해 주세요.

## 2. 테스트 방법 및 결과
- 동작 확인을 위해 어떤 테스트를 했는지, 결과는 어땠는지 적어 주세요.

## 3. 참고 자료/이슈
- 관련 문서, 참고한 이슈/PR 등이 있다면 링크와 함께 적어 주세요.

---

### 체크리스트
- [ ] 코드가 빌드/테스트를 통과함
- [ ] 팀 컨벤션(커밋, 브랜치, 코드 스타일 등)을 따름
- [ ] 리뷰어가 이해하기 쉽도록 설명을 충분히 작성함
